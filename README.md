# 🔍 Suspicious URL Detection System

## 📌 Project Description
This project is a basic cybersecurity tool built using Python and Streamlit. It helps detect suspicious URLs by analyzing patterns commonly used in phishing attacks.

## 🚀 Features
- Detects special characters like @, -, etc.
- Identifies URLs containing IP addresses instead of domain names
- Provides reasons why a URL is marked as suspicious
- Simple and user-friendly interface using Streamlit

## 🛠️ Technologies Used
- Python
- Streamlit
- Regular Expressions (re module)

## ▶️ How to Run
1. Install Streamlit:
2. Run the project:streamlit run suspicious_url_detector.py
